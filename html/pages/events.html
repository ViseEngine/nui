事件映射

事件映射模块是为了解决jQuery绑定事件时重复代码过多，缺少复用性等问题。

使用方法

使用前需先作为依赖引入

模块通过jQuery中的on来代理绑定事件，只接收一个参数，该参数为对象，提供了3个属性

属性

# element

类型：{jQuery Object}

说明：事件代理元素

# mapping

类型：{Object}

说明：设置事件映射

# callback

类型：{Object}

说明：设置事件回调
